
# MarTerra — Vite + Tailwind Starter (Ready to Deploy)

## Run locally
```powershell
cd marterra-site
npm install
npm run dev
```

## Deploy to Vercel (recommended)
1. Push to GitHub.
2. In Vercel, 'Add New Project' → import repo.
3. Build: `npm run build`, Output: `dist`.

## Notes
- All media expected under `public/assets/`.
- Paths use `/assets/...` in the code.
- Tailwind configured via `tailwind.config.js` and `postcss.config.js`.
